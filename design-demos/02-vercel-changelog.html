PR-1

全局微动效（transition + duration）

+0.4
type 工程 hours 0.5h risk 无 w1-mon

所有交互元素（按钮 / 面板 / 抽屉 / toast）加 150–200ms 颜色与位移过渡，定义 --ease-out-expo 全局常量。

实现要点

  • components/ActivityBar.tsx 加 transition-colors
  • ConsoleLayout.tsx drawer / sheet 加 transition-transform duration-300
  • ToastViewport.tsx 用 spring 缓动
  • globals.css 定义 --ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1)

回滚

移除对应 utility class 即可，零风险变更。

验收

  • 点击 ActivityBar 图标无闪烁
  • Toast 弹入 / 消失有 200ms 淡入淡出
  • prefers-reduced-motion 下禁用所有动画
